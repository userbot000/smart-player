# ✅ מערכת עדכונים פשוטה - ללא סיבוכים!

## 🎯 מה עשינו

מערכת עדכונים **פשוטה וישירה**:
1. בודקת ב-GitHub אם יש tag חדש
2. אם כן - מציגה הודעה
3. המשתמש לוחץ "הורד עדכון" - נפתח דפדפן
4. המשתמש מוריד ומתקין ידנית

**זהו! אין חתימות, אין סיבוכים, פשוט עובד.**

```
   ↓

   ↓
4. אם יש עדכון → הודעה
   ↓
5. משתמש לוחץ "התקן"
   ↓
6. מוריד .exe מ-GitHub
   ↓
7. מריץ התקנה
   ↓
8. אפליקציה נסגרת ונפתחת מחדש
   ↓
9. ✨ גרסה חדשה!
```

## 📦 יצירת Release

```bash
# 1. עדכן גרסה
# src-tauri/tauri.conf.json: "version": "0.2.0"
# src-tauri/Cargo.toml: version = "0.2.0"

# 2. Commit
git add .
git commit -m "Release v0.2.0"
git push

# 3. צור Tag
git tag v0.2.0
git push origin v0.2.0

# 4. GitHub Actions יבנה אוטומטית
# 5. Release ייווצר עם קובץ .exe
# 6. המערכת תזהה אותו אוטומטית!
```
