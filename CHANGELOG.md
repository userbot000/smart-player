# Changelog

כל השינויים המשמעותיים בפרויקט יתועדו כאן.

## [Unreleased]

### ✨ תכונות חדשות

#### 🎵 נגן
- **מהירות השמעה** - אפשרות לשנות מהירות השמעה (0.5x - 2x)
- **סימניות** - אפשרות להוסיף סימניות בשירים
- **תור חכם משופר** - אלגוריתם שלומד את ההעדפות שלך

#### 🎨 ממשק משתמש
- **RTL מלא** - התאמה מלאה לעברית
  - כפתורי ניווט הפוכים (הבא/קודם)
  - זמנים הפוכים (סיום/נוכחי)
- **סליידרים בתכלת** - החלק שעבר בשיר ובעוצמת קול בצבע תכלת
- **ערכות נושא** - בהיר/כהה/אוטומטי עם צבעי הדגשה

#### 📚 ניהול ספרייה
- **חיפוש משופר** - חיפוש בשירים, אמנים, אלבומים ותיקיות
- **סריקת תיקיות משופרת** - progress bar בזמן אמת לתיקיות גדולות
- **תצוגת תיקיות היררכית** - ניווט קל בתתי-תיקיות
- **רשימות השמעה** - יצירה וניהול של רשימות השמעה מותאמות אישית

#### 🛠️ כלים
- **יצירת רינגטונים** - חיתוך שירים ליצירת רינגטונים
- **עריכת מטאדאטה** - עריכת תגיות ID3 ותמונות אלבום
- **גישה מהירה** - אפשרות לעבור לכלים ישירות מתפריט השיר

#### 📱 Android
- **Workflow אוטומטי** - בניית APK אוטומטית ב-GitHub Actions
- **תמיכה מרובת ארכיטקטורות** - ARM64, ARMv7, Universal

### 🔧 שיפורים

#### ביצועים
- **קריאת מטאדאטה מהירה** - רק 256KB לכל קובץ במקום הקובץ המלא
- **עיבוד מקבילי** - עיבוד קבצים בקבוצות
- **Progress tracking** - משוב בזמן אמת על התקדמות

#### חוויית משתמש
- **הודעות ברורות** - Toast messages עם אייקונים
- **אינדיקטורים ויזואליים** - progress bars מפורטים
- **ניווט משופר** - breadcrumbs בתצוגת תיקיות

### 🐛 תיקוני באגים
- תיקון זיהוי Tauri בפרודקשן
- תיקון חיפוש בתיקיות - עכשיו מחפש גם בשמות תיקיות
- תיקון תצוגת תיקיות ברמה 3+
- תיקון שחזור מיקום בשיר
- תיקון פתיחת קבצים חיצוניים

### 📦 Build & Deploy
- **GitHub Actions Workflows**:
  - Build אוטומטי ל-Windows
  - Build אוטומטי ל-Android (עם אתחול)
  - Release אוטומטי עם tags
- **תיעוד מלא**:
  - מדריך מהיר לאנדרואיד
  - מדריך התקנה מקומית
  - Checklist מוכנות
  - מדריך יצירת releases

### 🔒 אבטחה
- חתימת קוד ל-Windows (אופציונלי)
- הרשאות Android מוגדרות

## פורמט

הפרויקט עוקב אחר [Semantic Versioning](https://semver.org/).

סוגי שינויים:
- `Added` - תכונות חדשות
- `Changed` - שינויים בתכונות קיימות
- `Deprecated` - תכונות שיוסרו בעתיד
- `Removed` - תכונות שהוסרו
- `Fixed` - תיקוני באגים
- `Security` - תיקוני אבטחה
